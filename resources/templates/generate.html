When this URI is accessed with an AJAX request, the server generates a graph with satellites in (pseudo) random locations, dispersed over earth at altitudes between 300 and 700 km. The graph is presented as a hashmap and sent from the server to the frontend in JSON format. Below is a satellite graph generated by the server. The data representation has been prettified compared to the raw JSON string.

Refreshing this page triggers the generation of a new dataset. The amount of generated satellites can be controlled by appending a query to the URL, for example generate?satelliteCount=100. If no query is given, the satellite count defaults to 20


Nodes

Nodes are given as hashmaps, with the keys :name and :pos. The position is a vector with the values x, y and z, where the origin is at the center of the earth. Below are the nodes in raw format.

Edges

Edges are given as vectors, where indexes 0 and 1 contain the nodes which are connected and index 2 contains the float value corresponding to the distance between these nodes measured in kilometers. Index 3 is an object containing only one key :is_solution_path, which maps to true if the corresponding edge is part of the shortest path between nodes named "START" and "END", else false.
